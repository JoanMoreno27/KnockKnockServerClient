Proyecto simple con la finalidad de probar a desarrollar por primera vez un programa simple que utilice
Sockets para hacer una tarea servidor cliente.

En este caso el programa será un chat en el que dependiendo de las respuestas de KnockClient las respuestas
de el servidor KnockServer serán diferentes, en este caso con respuestas de chistes de toc toc en tres idiomas
Catalán, Castellano e Inglés.
